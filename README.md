## 기술 스택

**`Backend`**: **Java, Spring Boot**

**`Database`**: **MySQL, Data JPA, QueryDSL**

**`AWS`**: **EC2, RDS, S3**

**`외부API`** : **한국어 기초사전 API, 소셜로그인(NAVER, KAKAO) API**

**`Etc`**: **Swagger**
